# example-mkdocs-mike
An example project to demonstrate mkdocs + mike documentation publication to GitHub pages

It generates a documentation website that is accessible at the following locations:

* Default version: https://joffrey-bion.github.io/example-mkdocs-mike/
* Dev version (from `main`): https://joffrey-bion.github.io/example-mkdocs-mike/dev/
* Release versions (from `releases/*`): https://joffrey-bion.github.io/example-mkdocs-mike/0.9/
